# MyPanel Home Assistant Add-ons

Custom add-on repository for the MyPanel ESP32-S3 smart home touch panel.

## Installation

1. In Home Assistant, go to **Settings → Add-ons → Add-on Store**
2. Click **⋮** (top right) → **Repositories**
3. Paste this URL and click **Add**:
   ```
   https://github.com/ackrosee-wq/mypanel-ha-addon
   ```
4. Find **MyPanel Bridge** in the store and click **Install**
5. Start the add-on — it auto-detects your MQTT broker from the Mosquitto add-on

## Requirements

- Home Assistant OS or Supervised
- Mosquitto MQTT Broker add-on (installed and running)
